Проект является реализацией домашнего задания по теме NLP.

1. Данные представляют из себя отзывы к различным смартфонам и были собраны с сайта 'e-catalog'.
    У каждого из отзывов есть метка (sentiment), 
    выбранная пользователем из 4 возможных, от негативного к позитивному.
    Код сбора и данных содержиться в ноутбуке 'collect_phone_reviews.ipynb'.
    
2. В остальных ноутбуках содержаться решения задачи определения метки отзыва с помощью методов NLP и
    машинного обучения. Есть четыре экспериментальных ноутбука, отличающиеся лишь предобработкой данных:
        1. 'classification_phone_reviews_2_classes_full_data' - Разбиение на 2 класса с сохранением дисбаланса.
        2. 'classification_phone_reviews_2_classes_undersampling' - - Разбиение на 2 класса c балансировкой.
        3. 'classification_phone_reviews_4_classes' - Разбиение на 4 класса с сохранением дисбаланса.
        4. 'classification_phone_reviews_4_classes_balanced' - Разбиение на 4 класса c балансировкой.
